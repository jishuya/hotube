# 1.계획

1. 소프트웨어 종류: 개인용 youtube 사이트 
2. 소프트웨어 목표: 비공개 유부트 링크의 영상들을 가지고 유투브와 똑같이 생긴 사이트 만들기 
3. 사용 상황: 가족들이 PC, 태블릿, 휴대폰을 이용하여 사이트에 접속하여 신청

# 2. 요구사항분석

**1. Homepage (홈페이지)**

- 가족 전용의 개인 유튜브 스타일 홈페이지입니다. 최신 영상이 가장 먼저 표시되며, 유튜브와 유사하게 동영상 두 줄, 쇼츠 한 줄이 반복되는 레이아웃. PC, 태블릿, 모바일에서 모두 잘 보이는 반응형 디자인. 일반 동영상은 16:9 숏츠(가로영상)는 9:16의 비율을 가지며 화면 크기(PC / 태블릿 / 모바일)에 따라 “너비”만 바뀌고, 높이는 비율로 따라가는 구조
- **세부 기능:**
    - **로고:** 페이지 상단 왼쪽에 위치합니다.
    - **동영상 검색창:** 페이지 상단 중앙에 위치하여 영상 검색 기능을 제공합니다.
    - **유저 아이콘:** 페이지 상단 오른쪽에 위치하여 사용자 관련 기능을 제공합니다.
    - **카테고리 탭 ("동영상", "쇼츠", "연도별"):** 탭 클릭 시 해당 카테고리에 맞는 영상만 필터링하여 보여줍니다.
    - **메인 영역:** 동영상 및 쇼츠 썸네일과 정보가 혼합되어 표시되며, 각 영상 설명에는 연관검색어(#바닷가 #가족여행 등)가 포함됩니다.

**2. Video Playback Page (동영상 상세/재생 페이지)**

- 홈페이지에서 특정 영상 카드를 클릭했을 때 열리는 동영상 상세 및 재생 화면
- **세부 기능:**
    - **간단한 헤더:** 상단에 로고와 검색창이 있는 간단한 헤더가 유지됩니다.
    - **큰 동영상 플레이어:** 중앙에 큰 동영상 플레이어 영역이 있어 영상을 재생합니다.
    - **영상 정보:** 플레이어 아래에 영상 제목, 업로드 날짜/연도, 그리고 키워드가 포함된 간단한 설명 텍스트가 표시됩니다.
    - **추천 영상 리스트:** 오른쪽 또는 아래쪽에 다른 추천 영상 카드 목록이 눈에 띄게 배치되어 있습니다.

**3. Admin Page(관리자 페이지)**

- 영상 관리(CRUD)를 위한 관리자 전용 대시보드 페이지.
- 관리자 페이지로 들어가기 위해서는 비밀번호를 입력해야함
- **세부 기능:**
    - **새 영상 등록 폼:** 제목, 유튜브 링크(URL), 타입("동영상"/"쇼츠"), 연도(드롭다운), 카테고리 태그, 키워드 입력 필드 및 "업로드" 버튼을 포함합니다.
    - **등록된 영상 목록 테이블:** 이미 등록된 영상들의 썸네일, 제목, 타입, 연도, 카테고리, 등록일 정보가 표시됩니다.
    - **수정/삭제 아이콘:** 각 영상 항목 옆에 수정 및 삭제 기능 아이콘이 제공됩니다.
